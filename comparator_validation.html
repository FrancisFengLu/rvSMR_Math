rvSMR comparator validation viewer

How five MR methods validate — and where they leave gaps
data sources: research_RARE.md · research_MR-CARV.md · research_AR_cisMR_comparators.md · VALIDATION_PLAN.md

Five Mendelian-randomization methods (RARE, MR-CARV, Wang–Kang, Patel–Lane–Burgess, cis-MRBEE) form the comparator landscape for rvSMR, a rare-variant single-gene MR method using Anderson–Rubin inference. Each paper sets a partial bar for what counts as “validation” in modern MR — some run weak-IV sweeps, some ship code, some use drug-target positive controls, none assemble a curated negative-control panel. This page walks each method's choices side-by-side so the gaps rvSMR can credibly close are visible at a glance.

Headline insights

5 / 5 papers cite a coverage figure
… but only 2 / 5 actually show coverage vs $F$-statistic (Wang–Kang Fig 6, Patel–Lane–Burgess Fig 2). The other three skip the canonical sweep.
3 / 5 ship a GitHub repo with simulation code
RARE, Patel–Lane–Burgess, and cis-MRBEE publish runnable code. MR-CARV ships an R package without the simulation generator; Wang–Kang has no public repo at all.
0 / 5 use a curated negative-control panel
No comparator ships biologically implausible (gene×outcome) pairs as a calibration check. rvSMR's opening.

X2 = G %*% gamma2                                + eps      # secondary exposure

# theta drives uncorrelated horizontal pleiotropy (UHP)

Wang–Kang
Wang S., Kang H. (2022). Biometrics 78(4):1699. AR / Kleibergen / CLR for two-sample MR.
DOI arXiv 1909.06950 PMID 34213007 GitHub: not found

Generative model

# Two-sample, individual-level; Section 3 Eqs. 1/3
n1 = n2 = 100 000;  L = 100 instruments
G_j ~ Binomial(2, p_j),  p_j ~ Uniform(0.1, 0.9)
gamma_j in [ sqrt((r-0.5)/n1) , sqrt((r+0.5)/n1) ]   # local-to-zero IV strength
(U_i, eps_X_i, eps_Y_i) joint normal,  rho_endog = 0.1
X_i = G_i %*% gamma + eps_X_i
Y_i = beta * X_i + alpha_j' G_i + eps_Y_i
# r ~= first-stage F-statistic
# r in {1, 4, 16, 25, 50} -> the central weak-IV stress axis
# AR statistic: AR(beta_0) = (n1 + n2) * (gamma_hat' Sigma^-1 gamma_hat - ...)
# rejecting H_0 : beta = beta_0 when AR(beta_0) > chi^2_{L, 1-alpha}

Patel–Lane–Burgess
Patel A., Lane J., Burgess S. (2024). arXiv 2408.09868. Weak instruments in multivariable MR; AR + Kleibergen-OH.
arXiv 2408.09868 GitHub: ash-res/mvmr-weakiv

Generative model

# Two-sample summary; J = 4 instruments, K = 2 exposures (baseline)
n_X = n_Y = 5 000
gamma1 = c(1+xi, 1+xi, 1-xi, 1-xi) * 0.2 * n_X^(-1/2) * mu
gamma2 = c(1-xi, 1-xi, 1+xi, 1+xi) *       n_X^(-1/2) * mu
X_k = Z %*% gamma_k + V_k
Y   = theta1 * X1 + theta2 * X2 + U                    # truth (1, 0)
Cor(U, V1) = -0.6, Cor(U, V2) = +0.6, Cor(V1, V2) = 0.3
Z ~ MVN; off-diagonal correlation up to 0.4
# mu = unconditional strength, xi in [0,1] = conditional strength
# F_min crosses 10 in the swept cells

Weakest $F$-stat regime each method actually tested

Only Wang–Kang and Patel–Lane–Burgess explored a genuinely weak-IV regime. RARE, MR-CARV, and cis-MRBEE never push $F$ below 10. rvSMR's planned floor: $F \approx 0.5$.

RARE / MR-CARV / cis-MRBEE bars represent the implicit floor of the strongest cell they ran (no $F$ axis tested). Values: Wang–Kang $r=1$ ≈ $F=1$; PLB $F_{\min}\approx 6$; cis-MRBEE implicit at $n=300$; MR-CARV / RARE not parameterized.

The canonical coverage figure SCHEMATIC

This is the figure both Wang–Kang (Fig 6) and Patel–Lane–Burgess (Fig 2 top row) make the centerpiece of their paper. $y$-axis: empirical 95% CI coverage from 500–1000 reps; $x$-axis: first-stage $F$ on log scale. Anderson–Rubin holds a flat line at 0.95. Wald / IVW collapse below $F=10$, dropping to ~50% at $F=1$. The sketch below shows the qualitative shape rvSMR-AR must reproduce on the rare-variant burden axis.

Sketched data points, not real simulation output. Real numbers will come from Track 1 of the validation plan: 1 000 reps per cell, $F \in \{0.5, 1, 2, 5, 10, 30, 100\}$, decision rule "rvSMR-AR passes if empirical coverage $\in [0.93, 0.97]$ everywhere."

rvSMR's plan

Four parallel validation tracks (from VALIDATION_PLAN.md §3)

Track 1 — Simulation harness with swept $F$

unblocked · primary

1000G-anchored generator with rare-variant mask structure and an explicit $F$ knob. Produces the canonical coverage-vs-$F$ figure across $F \in \{0.5, 1, 2, 5, 10, 30, 100\}$, $\beta \in \{0, 0.04, 0.1, 0.2, 0.4\}$, $K \in \{1, 2, 3, 5\}$ masks. Compares Wald, IVW, dIVW, MR-RAPS, Egger, PRESSO, RARE, classical SMR vs rvSMR-AR. 1 000 reps for power; 10 000 for Type-I.

Track 2 — Common-variant plumbing test

unblocked

End-to-end PCSK9 $\to$ LDL run on TenK10K Phase 1 common-variant cis-eQTL (Zenodo 17474113, real) $\times$ Genebass burden outputs. Engineering test: prove the pipe handles 14–23 GB files, real LD, real annotation. Not a methodology test.

Track 3 — Real rare-variant exposure

blocked · on Wei / Cuomo

Headline panel: 5 RCT-validated genes (PCSK9, ANGPTL3, APOC3, HMGCR, LPA) $\times$ 5 outcomes, $K = 3$ masks $\times$ 28 PBMC cell types. Plus 5 negative-control pairs (biologically implausible gene$\times$outcome) — the row no comparator has. Gated on Zenodo 17474113 rare-variant zips being populated (currently 214-byte placeholders).

Track 4 — Sub-sampling + Goodhart

unblocked (prototypes on Track 2)

Borrowed from Patel–Lane–Burgess. (a) Sub-sample Genebass to $n \in \{2\text{k}, 10\text{k}, 50\text{k}, 100\text{k}\}$ with truth fixed at full-sample point estimate; heatmap of CI shapes. (b) Goodhart figure: report coverage with and without an $F > 10$ pre-screen.

Track 5 — Annotation-class Q with pQTL anchors

unblocked

Dhindsa 2023 UKB-PPP rare-variant pQTL anchors normalize class-specific burden-on-outcome estimates; Cochran-Q across the three mask classes (pLoF / mis:LC / reg). Expected $Q$ p-value $> 0.05$ for validated genes.

Deliverables

scaffold + CI gate

inst/sim/generate.R; tests/testthat/test-coverage_vs_F.R — CI gate that fails if rvSMR-AR coverage drops below 0.93 anywhere; figures/coverage_vs_F.qmd (single-command Quarto for the headline figure); Zenodo deposit + DOI.
