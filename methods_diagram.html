RARE · Bayesian MVMR

RARE — 多曝光贝叶斯 + CHP/UHP 分解

图 1 · 流水线 Pipeline

图 2 · 结构方程 Structural model

IV→曝露 信号$G\delta$ + CHP (共享上游通路)$G\kappa_x$ + 潜在混杂$U\psi_x$ + 噪声$\varepsilon$  =  $X_1$
IV→次要曝露$G\gamma_2$ + 噪声$\varepsilon$  =  $X_2$
想识别的因果效应$\beta_1 X_1$ + 次要曝露的因果效应$\beta_2 X_2$ + CHP (outcome 端)$G\kappa_y$ + UHP 不相关水平多效性$G\theta$ + 潜在混杂$U\psi_y$ + 噪声$\varepsilon$  =  $Y$
$G$ = 基因型矩阵 (1000 common + 20 rare); $U$ = 20 个 latent confounders; $\kappa$ 项产生 相关水平多效性 CHP (correlated horizontal pleiotropy, 通过共享上游通路同时影响 X 与 Y); $\theta$ 项产生 不相关水平多效性 UHP (uncorrelated horizontal pleiotropy, IV 绕过 X 直接打到 Y). RARE 的本质就是: 把 $\kappa$ 和 $\theta$ 当成可估的随机效应, 用 Bayesian Gibbs 把它们和 $\beta_1$ 一起采样.
RARE 一句话总评: Bayesian + MVMR + CHP/UHP 分解; 强项是先验对弱 IV 的隐式正则化 (Normal prior 把估计往 0 拉, 弱 IV 时 bias 比 IVW 小); 弱项是 credible interval ≠ frequentist coverage, 而且没有 F 扫描, 没人能保证它在 $F < 1$ 时 type-I rate 仍然 0.05.
MR-CARV · IVW + burden as IV

MR-CARV — 把稀有 burden 当作额外 IV 拼进 IVW

图 1 · 流水线 Pipeline

图 2 · 工作模型 Working model

公共部分 (common SNP 当 IV):
公共 IV → Y$\hat\beta_{y,i}$ = 因果效应$\beta$ · 公共 IV → X$\hat\beta_{x,i}$ + 噪声$\varepsilon_i$ $i = 1,...,4$
稀有部分 (burden 当 IV):
burden → Y$\hat\beta_{y,j}$ = 因果效应 (同一 β)$\beta$ · burden → X$\hat\beta_{\text{burden},j}$ + 噪声$\varepsilon_j$ $j = 1,...,4$
堆叠成 8-向量 → 标准 IVW 估计:
$\displaystyle \hat\beta_{\text{IVW}} \;=\; \frac{\sum_{k=1}^{8} w_k \,\hat\beta_{x,k}\, \hat\beta_{y,k}}{\sum_{k=1}^{8} w_k \,\hat\beta_{x,k}^{2}}, \qquad w_k = 1/\mathrm{SE}(\hat\beta_{y,k})^2$
注意: MR-CARV 的核心动作就是 "把 burden 当 IV 拼进 IVW", 没有 Anderson–Rubin (AR), 没有 cell-type 分层, 也没有 over-identification 测试. IVW 假设强 IV (concentration parameter → ∞), burden 信号弱时它会 bias 向 0; 作者用 dIVW / MR-RAPS 来部分缓解, 但都是 "many weak instruments" 框架, 不是 weak-IV-robust.
MR-CARV 一句话总评: 把稀有 burden 当作额外 IV 喂进 IVW/dIVW/RAPS; 强项是简单 + 直接组合既有工具 (实现门槛极低); 弱项是仍假设强 IV, 没 AR, 而且 CCT 用错了地方 — 它在 (mask × MAF-weight) 同 gene 内部合并 (内部 p-values 高度相关), 而不是跨真正正交的轴.
