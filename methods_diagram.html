RARE · Bayesian MVMR

RARE — 多曝光贝叶斯 + CHP/UHP 分解

图 1 · 流水线 Pipeline

图 2 · 结构方程 Structural model

IV→曝露 信号$G\delta$ + CHP (共享上游通路)$G\kappa_x$ + 潜在混杂$U\psi_x$ + 噪声$\varepsilon$  =  $X_1$
IV→次要曝露$G\gamma_2$ + 噪声$\varepsilon$  =  $X_2$
想识别的因果效应$\beta_1 X_1$ + 次要曝露的因果效应$\beta_2 X_2$ + CHP (outcome 端)$G\kappa_y$ + UHP 不相关水平多效性$G\theta$ + 潜在混杂$U\psi_y$ + 噪声$\varepsilon$  =  $Y$
$G$ = 基因型矩阵 (1000 common + 20 rare); $U$ = 20 个 latent confounders; $\kappa$ 项产生 相关水平多效性 CHP (correlated horizontal pleiotropy, 通过共享上游通路同时影响 X 与 Y); $\theta$ 项产生 不相关水平多效性 UHP (uncorrelated horizontal pleiotropy, IV 绕过 X 直接打到 Y). RARE 的本质就是: 把 $\kappa$ 和 $\theta$ 当成可估的随机效应, 用 Bayesian Gibbs 把它们和 $\beta_1$ 一起采样.

3b · 逐步抽样配方 Step-by-step

  1. 抽 SNP 效应 (未缩放): delta ~ N(0, 1), gamma2 ~ N(0, 1), theta ~ N(0, 1), 每项 1,020 维.
  2. 抽 CHP 相关对 $(\kappa_x, \kappa_y)$: $\binom{\kappa_{x,j}}{\kappa_{y,j}} \sim \mathcal{N}_2\!\left(\mathbf{0},\, \begin{pmatrix}1&\rho_\kappa\\\rho_\kappa&1\end{pmatrix}\right)$, $j = 1,\dots,1020$.
  3. 抽混杂载荷 $(\psi_x, \psi_y)$: $\binom{\psi_{x,k}}{\psi_{y,k}} \sim \mathcal{N}_2(\mathbf{0},\,I_2)$, $k = 1,\dots,20$.
  4. 抽公共基因型 $G_c$: 先抽连续 multivariate-normal 矩阵 $\tilde G_c \sim \mathcal{N}_n(\mathbf{0}, \Sigma_c)$, 协方差 $\Sigma_{c,ij} = 0.2^{|i-j|}$ (指数衰减 LD). 然后按每 SNP 独抽的 $\mathrm{MAF}_j \sim \mathrm{Unif}(0.05, 0.5)$ 切分位: 
    0 if rank < MAF², 1 if rank < MAF·(2-MAF), 2 otherwise
    (即 Hardy-Weinberg AA/Aa/aa 比例的"分位映射"近似 Binomial$(2, \mathrm{MAF})$).
  5. 抽稀有基因型 $G_r$: 同上但 $\Sigma_{r,ij} = 0.5^{|i-j|}$ 且 $\mathrm{MAF}_j \sim \mathrm{Unif}(0.005, 0.01)$.
  6. 拼接 $G = [G_c \mid G_r]$, 维度 $n \times 1020$.
  7. 抽潜在混杂 $U$: $U_{ik} \sim \mathcal{N}(0,1)$ i.i.d., 形状 $n \times 20$.
  8. 计算总方差归一化因子 $k_{\text{scale}} = (\beta_1^2 + \beta_2^2 + 1) / (1 - \sum h^2)$.
  9. 缩放各 SNP 效应 使得每项在 $X_1$ 的解释方差正好等于目标 $h^2$: 
    delta  <- delta  · sqrt( h²_δ  · k_scale / (β₁² · var(G·delta)) )
kappax <- kappax · sqrt( h²_κx · k_scale / (β₁² · var(G·kappax)) )
gamma2 <- gamma2 · sqrt( h²_γ2 · k_scale / (β₂² · var(G·gamma2)) )
kappay <- kappay · sqrt( h²_κy · k_scale /       var(G·kappay)   )
theta  <- theta  · sqrt( h²_θ  · k_scale /       var(G·theta )   )
    (当对应 $h^2 = 0$ 时该 column 全置零, 见 if (h2_kappay == 0) G_kappay <- G %*% rep(0, nsnp).)
  10. 缩放混杂贡献 $U\psi$ 到固定 0.8 的方差: U_faix <- (U %*% faix) / sqrt(var(...)/0.8), 同理 $U\psi_y$.
  11. 组装 $X_1$: $$X_1 = G\delta + G\kappa_x + U\psi_x + \varepsilon_1,\quad \varepsilon_1 \sim \mathcal{N}\!\left(0,\,1 - \mathrm{Var}(U\psi_x)\right).$$
  12. 组装 $X_2$: $X_2 = G\gamma_2 + \varepsilon_2$, $\varepsilon_2 \sim \mathcal{N}(0,1)$.
  13. 构造稀有 PRS: 先在 summary panel 上跑单变量回归 $\hat{c}_j = \mathrm{fastSigLm}(X_1, G_j)$; 取后 20 个 (rare) 系数构成 PRS = Gr · coef_r, 再标准化到方差 1.
  14. 抽异方差 Y 噪声: $\varepsilon_{Y,i} \sim \mathcal{N}(0, \sigma_i^2)$, $\sigma_i = \dfrac{\mathrm{PRS}_i - \mathrm{PRS}_{\min}}{\mathrm{PRS}_{\max} - \mathrm{PRS}_{\min}} + 0.05$.
  15. 组装 $Y$: $$Y = \beta_1 X_1 + \beta_2 X_2 + G\kappa_y + G\theta + U\psi_y + \varepsilon_Y\,\sqrt{1 - \mathrm{Var}(U\psi_y)}.$$
  16. 计算 summary stats (calculation panel): 对每个 SNP $j$ 单独 OLS 回归: $\hat\beta_{x_1,j} = \mathrm{fastSigLm}(X_1, G_j)$, 同理 $\hat\beta_{x_2,j}, \hat\beta_{\text{PRS},j}, \hat\beta_{y,j}$, 每个带 SE.
  17. 构造 MV-MR 输入: $\hat B_x = [\hat\beta_{x_1}, \hat\beta_{x_2}, \hat\beta_{\text{PRS}}]$ (1020 $\times$ 3); $\hat\beta_y$ (1020 $\times$ 1). 喂给 adapt(...) Gibbs sampler (1000 iter, burn-in = 500). naive 版本仅用 $[\hat\beta_{x_2}, \hat\beta_{\text{PRS}}]$ 两列.
  18. 记录 Type-I / bias: 95% credible interval 是否覆盖 0; $\hat\beta_1 - \beta_1^{\text{true}}$. 跨 300 reps 平均.

RARE 一句话总评: Bayesian + MVMR + CHP/UHP 分解; 强项是先验对弱 IV 的隐式正则化 (Normal prior 把估计往 0 拉, 弱 IV 时 bias 比 IVW 小); 弱项是 credible interval ≠ frequentist coverage, 而且没有 F 扫描, 没人能保证它在 $F < 1$ 时 type-I rate 仍然 0.05.
MR-CARV · IVW + burden as IV

MR-CARV — 把稀有 burden 当作额外 IV 拼进 IVW

图 1 · 流水线 Pipeline

图 2 · 工作模型 Working model

公共部分 (common SNP 当 IV):
公共 IV → Y$\hat\beta_{y,i}$ = 因果效应$\beta$ · 公共 IV → X$\hat\beta_{x,i}$ + 噪声$\varepsilon_i$ $i = 1,...,4$
稀有部分 (burden 当 IV):
burden → Y$\hat\beta_{y,j}$ = 因果效应 (同一 β)$\beta$ · burden → X$\hat\beta_{\text{burden},j}$ + 噪声$\varepsilon_j$ $j = 1,...,4$
堆叠成 8-向量 → 标准 IVW 估计:
$\displaystyle \hat\beta_{\text{IVW}} \;=\; \frac{\sum_{k=1}^{8} w_k \,\hat\beta_{x,k}\, \hat\beta_{y,k}}{\sum_{k=1}^{8} w_k \,\hat\beta_{x,k}^{2}}, \qquad w_k = 1/\mathrm{SE}(\hat\beta_{y,k})^2$
注意: MR-CARV 的核心动作就是 "把 burden 当 IV 拼进 IVW", 没有 Anderson–Rubin (AR), 没有 cell-type 分层, 也没有 over-identification 测试. IVW 假设强 IV (concentration parameter → ∞), burden 信号弱时它会 bias 向 0; 作者用 dIVW / MR-RAPS 来部分缓解, 但都是 "many weak instruments" 框架, 不是 weak-IV-robust.

3b · 逐步抽样配方 Step-by-step (一个仿真样本 $i$)

  1. 抽 8 条 SNP 序列: 用 sim1000G 从 1000 Genomes 真实单倍型重抽 $N = 10{,}000$ 个个体在 chr 15–22 上各 10 kb 区段的基因型. chr 15–18 每条只保留 1 个 MAF $> 0.1$ 的 SNP $\Rightarrow$ 公共 IVs $G_1, \dots, G_4$. chr 19–22 每条保留所有 MAF $\in (0.001, 0.05)$ 的 SNP, 取 50 个 $\Rightarrow$ rare-set $G_{j,1}, \dots, G_{j,50}$ 对 $j = 1, \dots, 4$.
  2. 抽功能注释 $A$: 生成 10 个独立 $\mathcal{N}(0,1)$ 列, 随机选 5 列作为"真正相关"的注释 $A_1, \dots, A_5$. (待核实: 这一步在 v1 method 文本里描述, 数值具体维度可能略有不同.)
  3. 抽因果指示 $\varphi_{j\ell}$: 对 burden $j$ 内每个稀有变体 $\ell$, 取 Bernoulli$\left(\mathrm{expit}\!\left(b_0 + \log 5 \cdot \sum_i A_{j\ell,i}\right)\right)$. $b_0$ 网格搜索, 使整 set 内平均 $\varphi = 1$ 比例 $\approx 15\%$ 或 $\approx 35\%$.
  4. 抽方向: 给被选中的 $\varphi_{j\ell} = 1$ 变体里, 按正效应比例 $\pi^+ \in \{1, 0.8, 0.5\}$ 决定 $\mathrm{sign}(\alpha_{j\ell}) = +1$ 的比例.
  5. 计算 per-variant 效应: 公共 $\alpha_i = 0.5$ (固定). 稀有 $\alpha_{j\ell} = \mathrm{sign} \cdot 0.5 \cdot \lvert\log_{10}\mathrm{MAF}_{j\ell}\rvert$.
  6. 组装曝露 $X_i$ (sample $i$): $$X_i = \sum_{k=1}^4 \alpha_k G_{i,k} \;+\; \sum_{j=1}^4 \sum_{\ell=1}^{50} \varphi_{j\ell}\, \alpha_{j\ell}\, G_{i, j\ell} \;+\; \varepsilon_{X,i}$$ 连续 $X$: $\varepsilon_{X,i} \sim \mathcal{N}(0, 1)$. 二元 $X$: $P(X_i = 1) = \mathrm{expit}(\sum \alpha G)$, 即把上式截至噪声前喂 logit.
  7. 组装结果 $Y_i$: 连续 $Y$: $Y_i = \beta X_i + \varepsilon_{Y,i}$, $\varepsilon_{Y,i} \sim \mathcal{N}(0,1)$. 二元 $Y$: $P(Y_i = 1) = \mathrm{expit}(\beta X_i)$. 注意: 没有 $U$, 没有 $G \to Y$ 直接路径 (即仿真不引入 horizontal pleiotropy, 不同于 RARE).
  8. 跑 GWAS summary stats: 对每个 $G_k$ 单 SNP 回归 $X$ $\to$ $(\hat\beta_{x, k}, \mathrm{SE})$, 同理 $Y$. 4 个公共 SNP 各 1 行.
  9. 跑 burden test (per mask × per MAF weight): 对每个 burden $j$, STAARpipeline 输出 $(\hat\beta_{\text{burden}, j}, \mathrm{SE})$, 用 Beta$(1,25)$ 或 Beta$(1,1)$ 作为 SNP 内权重. 共 $4 \times 2 = 8$ 个 burden 估计.
  10. 拼成 8 维 IV 向量: $\hat\beta_x = [\hat\beta_{x,1..4} \mid \hat\beta_{\text{burden},1..4}]$, $\hat\beta_y$ 同理. 喂 IVW / dIVW / MR-RAPS.
  11. 用 CCT 合并: 对每个 (mask, weight) 组合各跑一次, 把 p-values 用 Cauchy combination test 聚合为单一报告 p.
  12. Type-I / power 累积: $\beta = 0$ 时跑 10,000 reps, $\beta \neq 0$ 时跑 1,000 reps.

3c · 一次仿真的数值示例 Worked example

假设 cont/cont 设定, $\beta = 0.04$, 因果变体比例 $\approx 35\%$, 正效应比例 $\pi^+ = 80\%$, burden 内权重 Beta$(1, 25)$. 看第 $i = 1$ 个样本.

数值为合理示例 (consistent with the recipe), 不是真实 seed=1 output.
公共 SNP 1 (chr 15): $\mathrm{MAF}_1 = 0.27$ (real 1000G 序列)
$G_{1,1} = 1$ (heterozygous)
$\alpha_1 = 0.5$ (固定)
$\Rightarrow \alpha_1 G_{1,1} = 0.5$
公共 SNP 2,3,4 类似 ⇒ $\sum_{k=1}^4 \alpha_k G_{1,k} \approx 1.0$
Rare burden 1 (chr 19): $\mathrm{MAF}_{1,1} = 0.003$ ⇒ $\alpha_{1,1} = 0.5 \cdot 2.52 = 1.26$ (取 + 号, 因 $\pi^+ = 80\%$)
$\varphi_{1,1} = 1$ (Bernoulli; 35% 因果)
$G_{1, 1{:}1} = 0$ (稀有 ⇒ 极大概率 0)
$\Rightarrow \varphi_{1,1} \alpha_{1,1} G_{1,1{:}1} = 0$
假设 burden 1 内仅 SNP 17 携带 1 个 minor allele: $\mathrm{MAF}_{1,17} = 0.012$ ⇒ $\alpha_{1,17} \approx 0.96$; $\varphi_{1,17} = 1$; $G_{1, 1{:}17} = 1$ ⇒ 贡献 $\approx 0.96$
全部 4 个 burden 在样本 1 上 ⇒ $\sum_j \sum_\ell \varphi \alpha G \approx 1.1$
$\varepsilon_{X,1} \sim \mathcal{N}(0,1) \approx -0.3$
$X_1 \approx 1.0 + 1.1 - 0.3 = 1.8$
$\varepsilon_{Y,1} \sim \mathcal{N}(0,1) \approx 0.2$
$Y_1 = 0.04 \cdot 1.8 + 0.2 \approx 0.272$
注: 多数个体的 burden 求和接近 0 (rare-by-design); 信号在尾部少数携带者上集中 — 这正是 burden test 比单 SNP test 有优势的原因.
MR-CARV 一句话总评: 把稀有 burden 当作额外 IV 喂进 IVW/dIVW/RAPS; 强项是简单 + 直接组合既有工具 (实现门槛极低); 弱项是仍假设强 IV, 没 AR, 而且 CCT 用错了地方 — 它在 (mask × MAF-weight) 同 gene 内部合并 (内部 p-values 高度相关), 而不是跨真正正交的轴.
